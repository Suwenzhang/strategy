# A股股票推荐策略

基于Tushare数据，结合技术分析和基本面筛选的股票推荐系统。

## 功能特性

### 技术面分析 (60%权重)
- **均线系统**: 5日、10日、20日、60日均线排列判断
- **RSI指标**: 超卖/超买区域识别
- **MACD指标**: 金叉/死叉信号
- **布林带**: 价格相对位置分析
- **成交量分析**: 量价配合判断
- **涨跌幅控制**: 温和上涨/回调买入

### 基本面分析 (40%权重)
- **ROE (净资产收益率)**: 盈利能力评估
- **毛利率**: 产品竞争力
- **营收增长率**: 成长性评估
- **净利润增长率**: 利润增长质量
- **资产负债率**: 财务健康度

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 运行全市场扫描

```bash
python stock_recommendation.py
```

### 2. 分析指定股票

修改 `stock_recommendation.py` 中的 `main()` 函数：

```python
# 取消注释以下代码分析指定股票
target_stocks = ['000001.SZ', '600000.SH', '600519.SH', '300750.SZ']
stock_list = strategy.pro.stock_basic(ts_code=target_stocks,
                                      fields='ts_code,symbol,name,area,industry,list_date')
results = strategy.scan_market(stock_list=stock_list, top_n=10)
```

## 推荐等级说明

| 等级 | 评分 | 说明 |
|------|------|------|
| ★★★★★ 强烈推荐 | 75+ | 技术面和基本面双优 |
| ★★★★ 推荐 | 60-74 | 综合条件良好 |
| ★★★ 谨慎关注 | 45-59 | 可关注但需谨慎 |
| ★★ 观望 | 30-44 | 不建议操作 |
| ★ 不推荐 | <30 | 条件不满足 |

## 注意事项

1. **风险提示**: 本策略仅供参考，不构成投资建议。股市有风险，投资需谨慎。

2. **API限制**: Tushare免费版每分钟调用次数有限，全市场扫描可能需要较长时间。

3. **数据时效性**: 财务数据可能存在滞后，建议结合最新财报。

4. **参数调整**: 可根据个人偏好调整各指标的权重和阈值。

## 自定义调整

在 `StockRecommendationStrategy` 类中可以调整：

- `technical_score()`: 修改技术面评分规则
- `fundamental_score()`: 修改基本面评分规则
- `scan_market()`: 修改筛选条件和返回数量
